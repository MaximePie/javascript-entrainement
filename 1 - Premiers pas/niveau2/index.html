Comment charger un script JS depuis un fichier HTML

  • Créer un fichier script.js dans le dossier niveau2
  • Y placer le contenu de la balise script présente dans index.html
  • Dans index.html, modifier la balise script de manière à ce qu'elle importe le script.js récemment créé
  • Relancer la page index.html et vérifier qu'on ait bien une alerte, et un message dans la console
