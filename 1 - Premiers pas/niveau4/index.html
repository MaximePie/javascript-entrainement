Comment ajouter un EventListener en Javascript ?

Exemples d'utilisations de EventListener

Voici un bouton qui ne fait rien.

  • Dans script.js, ajouter un EventListener onClick de manière à ce que lorsque l'on clique sur le bouton, une alerte "Bonjour !" se déclenche.
  • Dans index.html, créer un deuxième bouton d'id "affichage" qui contient le texte de votre choix
  • Dans script.js, ajouter un EventListener onclick sur ce bouton de manière à ce que lorsque l'on clique sur le bouton, on affiche "Il fait beau aujourd'hui." dans la console
