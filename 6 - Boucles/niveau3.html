Exercice 1 - while

#tantque

Créer une boucle while de manière à ce que :

  • L'application demande un texte à l'utilisateur via prompt
  • Tant que l'utilisateur saisit une valeur différente de "suuuper", demander à nouveau le texte à l'utilisateur
  • Si l'utilisateur saisit le texte "suuuper", alors sortir de la boucle, et afficher dans la console le texte suivant : "Vous avez craqué le mot de passe, woaw !"

Ressource

Boucles while en JS

Astuce

Bloqué.e ? Vérifiez que votre boucle ne s'exécute pas à l'infini.

Si votre boucle s'exécute à l'infini, alors trouvez un moyen de satisfaire la condition de sortie de la boucle.

Exemple : while(1 === 1) { console.log("Je suis bloqué") }

"Tant que 1 est égal à 1, afficher "Je suis bloqué".

Cette boucle est donc infinie, car 1 sera toujours égal à 1

Astuce

Pour réussir cet exercice, vous pouvez vous mettre à la place d'un ordinateur.

"Tant que ça n'est pas égal à ça, alors je fais ça."

"Si ça est égal à ça, alors j'arrête de faire ça."

Astuce

Vous pouvez réfléchir au comportement que vous voulez répéter, ici, c'est redemander à l'utilisateur de saisir "Chat"

On peut également réfléchir à la condition de sortie. "Qu'est-ce qu'il me faut pour qu'il arrête de demander ? Que j'écrive Chat"

Autrement dit, "Que ma réponse soit égale à "Chat"".

Exercice 2

for, mais pour un tableau.

Il y a un tableau temperatures dans le script.

Créer une boucle for qui va aller de 0 à la longueur du tableau en progressant de 1, qui va répéter l'instruction suivante :

  • Afficher "L'élément N°X est Y"
  • X est la position de l'élément actuel dans le tableau
  • Y est la valeur de l'élément actuel dans le tableau

Résultat

Le résultat attendu est :

"L'élément N°0 est 40" "L'élément N°1 est 60" "L'élément N°2 est 80"

Attention ! Si on ajoute un élément dans le tableau, le code doit toujours afficher tous les éléments du tableau, sans avoir à modifier la limite du for.

Astuce

Si for permet de compter de 0 à 10, il est possible de compter de 0 à 12, 15, ou 18.

for (let compteur = 0; compteur < 18; compteur ++)

D'autre part, considérons un suuuper tableau notes égal à [2, 4, 8]

Pour accéder au premier élément, on peut écrire notes[0]

Pour accéder au deuxième élément, on peut écrire notes[1]

Donc pour accéder à tous les éléments, il suffit de remplacer l'index par le compteur, et voilà !
