Exercice 2 - forEach 2

Parcourir le tableau names de manière à ce que pour chaque élément :

  • On appelle la fonction direBonjour en lui passant en paramètre l'élément courant.

Astuce

Il est possible de faire tout ce qu'on veut dans une fonction de forEach !

Il est donc très facile d'appeler une autre fonction depuis cette fonction-là.

Astuce

forEach permet d'exécuter la même fonction de code pour chaque élément d'un tableau.
