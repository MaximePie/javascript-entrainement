Exercice 3 - Tableau de booléens

  • Créer un tableau qui contient les booléens true, false, true, true, false
  • Afficher le tableau dans la console.

Astuce, bon à savoir !

Les tableaux accueillent des valeurs du même type. Des numbers avec des numbers, des string avec des string, mais il est une mauvaise pratique de mélanger plusieurs types dans un tableau.

Par exemple : let laFoire = [2, "Lion", false] augmente grandement le risque d'anomalies dans le code !
