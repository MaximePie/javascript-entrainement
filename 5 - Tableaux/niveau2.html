Niveau 2 - Lire et modifier les éléments d'un tableau

Parce qu'avant de modifier quelque chose, il faut savoir ce que l'on veut modifier

Exercice 1 - Afficher un seul élément du tableau

Avec la console,

  • Afficher le deuxième élément du tableau longestHikingPathsInFrance
  • Afficher le troisième élément du tableau longestHikingPathsInFrance

Ressources

Afficher un élément d'un tableau

Astuce

La première position du tableau est 0

La deuxième position du tableau est 1

La troisième position du tableau est 2

... etc.

Autrement dit, en programmation, les tableaux commencent à 0

Astuce

Vous pouvez vérifier la position d'un élément grâce à la console.log

Exercice 2 - Modifier l'élément d'un tableau

Modifier le tableau "cats" de manière à ce que :

  • Son deuxième élément prenne la valeur "Bobby"
  • Son cinquième élément prenne la valeur "Juliette"
  • Son huitième élément prenne la valeur "Dandino"

Astuces

La position d'un élément dans un tableau s'appelle un index

Même si un index n'existe pas dans le tableau, il sera automatiquement créé
