Exercice 1

Conditions en Javascript

Créer une condition de manière à ce que :

  • Si la variable plat est égale à 'crevettes', afficher "Oh Oui !" dans la console
  • Sinon afficher "Oh non".

Exercice 2

Créer une condition de manière à ce que :

  • Si la variable temperature est supérieure à 0, afficher "Trop chaud !" dans la console, et modifier sa valeur pour qu'elle soit égale à -5.
  • Si la variable temperature est inférieure à 0, afficher "Trop froid là !" dans la console, et modifier sa valeur pour qu'elle soit égale à 0.
  • Si la variable temperature est égale à 0, afficher "Parfait" dans la console.
